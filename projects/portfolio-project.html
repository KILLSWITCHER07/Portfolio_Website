PROJECT: F1 PORTFOLIO

01. DESIGN PHILOSOPHY

This portfolio was built to reflect the precision and speed of Formula 1. The goal was to create a "digital cockpit" experience that showcases technical skills through a high-contrast, aggressive aesthetic.

02. UI/UX AERODYNAMICS

The site utilizes modern CSS Grid and Flexbox for a responsive layout that scales perfectly from desktop to mobile. Heavy focus was placed on CSS variables for consistent branding and backdrop-filters for a premium "glassmorphism" feel.
