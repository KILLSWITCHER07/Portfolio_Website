PROJECT: CAREER PREDICTOR

SYSTEM STATUS: PURPLE SECTOR

01. PROJECT DESCRIPTION

The Career Predictor App was engineered to bridge the gap between student skills and industry requirements. By utilizing a data-driven approach, the system analyzes user inputs to provide high-probability career path recommendations.

02. TECHNICAL USAGE

The backend utilizes Python for logic processing and SQL for managing large datasets of career parameters. The focus was to create a program that could store user answers and give them a career path.
